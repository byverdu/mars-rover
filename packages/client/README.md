# `@mars-rover/client package`

React application

## Usage

### Install dependencies

```shell
> yarn install
```

### Development

```shell
# UI will run on port 3000 and use a proxy to API's port 9000
# a mock-server will be running so no API is required for development

> yarn dev
```

### Tests

```shell
> yarn test:client:watch

# or

> yarn test:client
```

### Compile the app

```shell
> yarn build
